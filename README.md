# Invoice Automation — Experience Highlight

> A production-grade platform I designed and built. The source code and internal design are
> private; this page is a high-level summary of the work for professional reference.

I built an **automated invoice & tax-compliance platform** that turns supplier invoices and
bank statements into validated, filing-ready outputs — replacing slow, error-prone manual
processing with an automated, auditable workflow.

## What it delivers

- Automatic extraction of structured data from financial PDFs.
- Validation and reconciliation of that data against business and legal rules.
- Generation of compliant, filing-ready documents.
- A clean web interface for users, with secure multi-user access.
- Deployed to the cloud with monitoring and automated delivery.

## Skills demonstrated

**Backend:** Python · FastAPI · REST APIs · PostgreSQL
**AI / automation:** LLM-based document data extraction
**Cloud & DevOps:** GCP (Cloud Run) · Docker · CI/CD · monitoring & observability
**Frontend:** React · TypeScript

## My role

Sole designer and developer — from architecture and the data-extraction pipeline through to
